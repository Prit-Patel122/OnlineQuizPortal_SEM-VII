# 🧠 Online Quiz Portal

The **Online Quiz Portal** is a dynamic and user-friendly web application built using **ASP.NET Core MVC** that allows administrators to create and manage quizzes, and users (students) to participate in them online.  
This system automates the entire quiz-taking process — from question management to result evaluation — providing a secure and efficient platform for both learners and educators.

---

## 📋 Project Overview

In today’s digital learning environment, conducting quizzes and assessments online has become essential.  
This project aims to **simplify quiz management** by offering a robust web-based solution that enables:
- **Admins/Teachers** to create quizzes, manage questions, and track results.
- **Students** to take quizzes, view scores instantly, and review their answers.

The portal is designed to be **secure**, **scalable**, and **easy to use**, with a clean UI and real-time result calculation.

---

## 🚀 Key Features

### 👩‍🎓 Student Features
- ✳️ **User Registration and Login** — Students can sign up and securely log in.
- 📝 **Take Quizzes Online** — Attempt quizzes with a timer and auto-submit on timeout.
- ⏰ **Real-Time Countdown Timer** — Keeps track of remaining time per quiz.
- 📊 **Instant Results** — Automatically calculates and displays score after submission.
- 📜 **Result History** — Students can view their previous quiz performances and correct answers.
- 🌙 **Dark/Light Mode Support** — Optional theme toggle for better user experience.

### 🧑‍🏫 Admin/Instructor Features
- 🔐 **Admin Authentication** — Secure admin login panel for quiz management.
- ➕ **Create and Manage Quizzes** — Add new quizzes, define categories, and set difficulty levels.
- ✍️ **Add Questions** — Support for multiple-choice and true/false type questions.
- 🔄 **Edit or Delete Quizzes** — Full CRUD functionality for quizzes and questions.
- 👥 **Manage Users** — View, edit, or delete registered users.
- 📈 **View Performance Reports** — See quiz analytics and student performance summaries.
